# multiwfn | openEuler
Multiwfn is an extremely powerful program for realizing electronic wavefunction analysis, which is a key ingredient of quantum chemistry. Multiwfn is developed by Tian Lu and coworkers, and is free of charge, open-source, high-efficient, very user-friendly and flexible, it can perform a very wide range of wavefunction-based analyses.
